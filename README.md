> AppleScript applet made for Finder to open a terminal window in your current working directory (cwd).
![](/demo.gif)

## Install

```sh
git clone git@github.com:claymaks/tocwd.git
```

## Usage

```sh
command-drag `atcwd.app` to Finder toolbar
```
